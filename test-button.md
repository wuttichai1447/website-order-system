# 🧪 คู่มือการทดสอบปุ่ม "ดูรายละเอียด"

## ✅ สรุปการทำงาน

### 1. **ปุ่ม "ดูรายละเอียด"**
- **ตำแหน่ง**: 
  - หน้าแรก (`/`) - ส่วน "ผลงานเด่น"
  - หน้าผลงาน (`/portfolio`) - ทั้งส่วน "ผลงานเด่น" และ "ผลงานทั้งหมด"

- **การทำงาน**:
  - ใช้ `NuxtLink` ห่อ `UButton` เพื่อให้ navigation ทำงานถูกต้อง
  - เมื่อคลิกจะนำทางไปยัง `/portfolio/[id]` ตาม project.id

### 2. **หน้าแสดงรายละเอียด** (`/portfolio/[id]`)
- **Components ที่ใช้**:
  - `ProjectDetail` - แสดงรายละเอียดหลักของโปรเจกต์
  - `ProjectContent` - แสดงเนื้อหาที่แตกต่างกันตามแต่ละโปรเจกต์
  - `RelatedProjects` - แสดงผลงานที่เกี่ยวข้อง
  - `ProjectNavigation` - Navigation ระหว่างโปรเจกต์

### 3. **เนื้อหาที่แสดง**:
- ✅ Hero Image พร้อมชื่อและหมวดหมู่
- ✅ ข้อมูลโปรเจกต์ (ลูกค้า, ปีที่ทำ, ระยะเวลา)
- ✅ รายละเอียดโปรเจกต์ (description + longDescription)
- ✅ ฟีเจอร์หลัก (features)
- ✅ เทคโนโลยีที่ใช้ (technologies)
- ✅ Gallery ภาพเพิ่มเติม (พร้อม lightbox)
- ✅ Custom Content สำหรับแต่ละโปรเจกต์:
  - โปรเจกต์ ID 1: E-Commerce พิเศษ
  - โปรเจกต์ ID 2: Corporate Website พิเศษ
- ✅ ปุ่มสอบถามและดูผลงานอื่นๆ
- ✅ ผลงานที่เกี่ยวข้อง
- ✅ Navigation ระหว่างโปรเจกต์

## 🧪 วิธีทดสอบ

### ขั้นตอนที่ 1: เปิดเว็บไซต์
```
1. เปิดเบราว์เซอร์ไปที่ http://localhost:3000
2. รอให้หน้าเว็บโหลดเสร็จ
```

### ขั้นตอนที่ 2: ทดสอบจากหน้าแรก
```
1. เลื่อนหน้าลงไปหาส่วน "ผลงานเด่น"
2. คลิกปุ่ม "ดูรายละเอียด" ของโปรเจกต์แรก (E-Commerce Platform)
3. ตรวจสอบว่า:
   - URL เปลี่ยนเป็น /portfolio/1
   - แสดงรายละเอียดโปรเจกต์
   - มี Custom Content สำหรับ E-Commerce
```

### ขั้นตอนที่ 3: ทดสอบจากหน้าผลงาน
```
1. ไปที่หน้า /portfolio
2. คลิกปุ่ม "ดูรายละเอียด" ของโปรเจกต์ใดก็ได้
3. ตรวจสอบว่า:
   - URL เปลี่ยนถูกต้อง
   - แสดงรายละเอียดโปรเจกต์
   - มีเนื้อหาตามแต่ละโปรเจกต์
```

### ขั้นตอนที่ 4: ทดสอบ Features
```
1. ตรวจสอบ Gallery:
   - คลิกที่ภาพใน Gallery
   - ควรเปิด Lightbox
   - ใช้ปุ่มลูกศรหรือแป้นพิมพ์ (Arrow Left/Right) เพื่อเปลี่ยนภาพ
   - กด Escape เพื่อปิด Lightbox

2. ตรวจสอบ Navigation:
   - คลิกปุ่ม "โปรเจกต์ก่อนหน้า" หรือ "โปรเจกต์ถัดไป"
   - ควรนำทางไปยังโปรเจกต์อื่น

3. ตรวจสอบ Related Projects:
   - ดูผลงานที่เกี่ยวข้องด้านล่าง
   - คลิกเพื่อไปยังโปรเจกต์อื่น
```

## 🐛 หากพบปัญหา

1. **ปุ่มไม่ทำงาน**:
   - ตรวจสอบว่าใช้ `NuxtLink` ถูกต้อง
   - ตรวจสอบว่า `project.link` มีค่า

2. **ไม่แสดงรายละเอียด**:
   - ตรวจสอบ Console สำหรับ error
   - ตรวจสอบว่า project.id ตรงกับ route.params.id

3. **Custom Content ไม่แสดง**:
   - ตรวจสอบว่า project.id ตรงกับ template slot
   - ตรวจสอบว่า ProjectContent component ถูกใช้
